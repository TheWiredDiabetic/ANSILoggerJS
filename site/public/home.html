Make logging & debugging less boring.

ANSILoggerJS is a compact Javascript 'plugin' that removes the boringness from logging & debugging with it's color-enabled log features using the 'ANSI' format that works in most terminal outputs.

What's so good about it?

Global Support

Since ANSILogger gives away the fact it uses ANSI, it makes usage easier and supports most terminal outputs, such as Visual Studio Terminal.

Open Source

ANSILogger is built in one of the modern and common code languages, known as JavaScript making it open source and very easy to modify and customize for developers.

Browser Support

ANSILogger is built into two versions, one of which being supported for browsers and most other enviorments, which makes it easier to use both front and backend.
